# TryOnStylist - AI Virtual Fashion Platform 👗✨

TryOnStylist is a cutting-edge, AI-powered e-commerce and styling platform. It revolutionizes the online shopping experience by allowing users to virtually try on clothing, discover their perfect color palettes, and explore traditional regional styles.

## 🌟 Key Features

* **Virtual Try-On Studio:** Upload a photo and instantly see how garments from our shop look on you using advanced AI generation.
* **Web Try-On:** Paste a product link from popular e-commerce sites (like Amazon or Myntra) to extract the garment and try it on virtually.
* **Color Analysis & Matcher:** Use our interactive eyedropper tool to extract your skin, hair, and eye colors, discover your Seasonal Palette, and instantly check if a garment matches your unique colors.
* **Interactive Regional Map:** Explore the cultural tapestry of India and try on authentic traditional dresses from various states.
* **Dynamic E-Commerce Shop:** Browse a curated, modern clothing collection with a robust shopping cart and checkout system.
* **Personalized Dashboard:** A private hub to manage your generated fits, saved color profiles, and try-on history.
* **Smart Stylist Bot:** An integrated, interactive chat assistant to help you navigate the platform effortlessly.

## 🛠️ Tech Stack

* **Framework:** React + Vite
* **Routing:** React Router DOM
* **Styling:** Tailwind CSS + Font Awesome
* **API Communication:** Axios
* **State Management:** React Context API (Auth & Cart)
